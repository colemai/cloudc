To run this code make sure you have recent versions of rails, ruby and sqlite3 installed. These can all be installed with this package http://railsinstaller.org/en.

Clone the cloudc repo to your computer. Open the terminal, navigate into the cloudc repo. 

In the terminal run 'bundle install', then 'rake db:create' and then 'rake db:migrate'.

After this, run 'rails s'

Open your browser and navigate to http://localhost:3000/
